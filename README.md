# weatherlist

[설명]
- Seoul, London, Chicago, 세 도시의 날씨를 표시하는 앱
- 위 도시 순서대로 오늘을 포함한 5일간의 날씨를 표시
- 총 3개 도시의 각 5일간의 날씨를 한 화면에 표시
- 아래 예시 그림을 참고하여 구현
- 유닛 테스트 코드를 작성
